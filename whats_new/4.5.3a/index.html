What is new in WinBoard / XBoard 4.5.3a?

This version fixes a compile error for Winboard. The XBoard version is unchanged compared to 4.5.3.

  • Fix compile errors for Winboard.
  • Let Winboard Makefile build non-JAWS version by default.
